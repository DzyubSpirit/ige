An interactive graph editor.
============================

Key bindings:
-------------

- Action:
    - `a` Add node
    - `c` Link node
    - `d` Delete node
    - `w` Write to file
    - `o` Read from file
    - `q` Exit

- Movement:
    - `+` Zoom in
    - `-` Zoom out
    - `h` Move left
    - `j` Move down
    - `k` Move up
    - `l` Move right
    - `>` Rotate clockwise
    - `<` Rotate counterclockwise
